# Mehul Panicker Portfolio

Premium AI-Driven Digital Marketer Portfolio

## Quick Start

1. Extract this folder
2. Push to GitHub:
```bash
git init
git add .
git commit -m "Initial commit"
git remote add origin YOUR_GITHUB_URL
git push -u origin main
```

3. Deploy to Vercel:
- Go to vercel.com
- Click "New Project"
- Import your GitHub repo
- Click "Deploy"

Done! Your site will be live in 2 minutes.

## Or Deploy Instantly

```bash
npm install -g vercel
cd mehul-portfolio-clean
vercel
```

## Tech Stack

- Next.js 14
- TypeScript
- Tailwind CSS
- Framer Motion
- Recharts

## Customize

- Update social links in `components/Footer.tsx`
- Edit content in component files
- Change colors in `tailwind.config.js`

## Local Development

```bash
npm install
npm run dev
```

Open http://localhost:3000
